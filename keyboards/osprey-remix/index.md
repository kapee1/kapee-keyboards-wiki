## About

I really like how the original Osprey looks, but it is very difficult to assemble it yourself. This remix allows you to build it much easier and it still can be wireless! However, the assembly is not the easiest due to the size of the diodes.

## What changed? 
It uses SuperMini as controller. I used the original PCB as a template, so the outline barely changed to fit an MSK switch near the USB port. I had to slightly change the case USB hole and add a switch cutout. But the original plate still fits.
