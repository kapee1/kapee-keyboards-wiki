## Main Features

* Easy to build. Great for soldering practice.
* Supports all Cyrillic symbols in base layer.
* Compatible with most base-kit keycap sets. No 40s add-ons needed.
* Optional Panasonic EVQWGD001 or EC-11 encoder
* Plate also optional. You can build a keyboard without a plate to reduce cost.
* Variable bottom row with 3u spacebar support.
* Customizable magnet panels
* VIAL/QMK/[ZMK Firmware](https://github.com/aroum/zmk-soroka) by [aroum](https://github.com/aroum)
* Hotswap PCB available. Hotswap PCB uses different diodes. Check the BOM file for details.
